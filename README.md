## Installation

```bash
pip install -r requirements.txt
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
```

## Features
1. Login/Signup 
2. Search bar
3. Add to Cart
4. Payment Gateway
5. Sorted Category-wise
6. Access Control
